# GameZone Website

A modern browser gaming platform built using HTML5, CSS3, JavaScript, Bootstrap 5, and jQuery.

## Live Demo

https://games-zone-online.netlify.app/

## Features

- Responsive Design
- Mobile Friendly
- Local Storage High Scores
- Modern UI
- Multiple Browser Games

## Games Included

- Snake Game
- Memory Card Game
- Minesweeper
- 2048
- Flappy Bird
- Space Invaders
- Sudoku
- Quiz Master

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5
